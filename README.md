# Prompt Engineering — Zero to Mastery

A technically rigorous, career-grade prompt engineering course built with Next.js.

## Features

- 7 modules, 23 lessons
- Embedded video player per lesson
- Interactive quizzes with scoring
- Performance tracking & grade system
- Printable certificate of completion
- Fully mobile responsive

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy on Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

1. Push this repo to GitHub
2. Import into Vercel
3. Deploy — no config needed

## Tech Stack

- Next.js 14 (App Router)
- React 18
- Pure CSS-in-JS (no external UI libraries)
- Google Fonts (Syne + Instrument Serif + JetBrains Mono)
